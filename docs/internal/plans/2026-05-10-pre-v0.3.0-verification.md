```

`package.json` versions diverged from the registry: v0.2.8 is on `main` but never tagged, never published, no GitHub Release. PR #209 closed the milestone in the repo; the publish step never ran.

**Recommended action (User or Publisher Agent):**
```bash

Today's published 0.2.7 has the bin wrappers but not the new query verbs — running ADR-027 against 0.2.7 is partial, not complete. v0.2.8 publish is the gate.
